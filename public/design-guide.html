9. Adding the theme toggle button
index.html — paste into topbar + <head>
<!-- In <head> — BEFORE other CSS, prevents flash -->
<script src="/js/theme.js"></script>
<link rel="stylesheet" href="/css/theme.css" />

<!-- In .topbar, next to search input -->
<button class="theme-toggle" id="theme-toggle" title="Toggle theme">
  <svg class="icon-sun" ...>...</svg>   <!-- shown in light mode -->
  <svg class="icon-moon" ...>...</svg>  <!-- shown in dark mode -->
</button>

The full SVG code for both icons is already in design-guide.html's topbar above — copy it from there.
